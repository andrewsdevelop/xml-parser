# xml-parser

## Установка проекта

```
npm install
```

### Запуск в режиме разработчика

```
npm run serve
```

### Сборка билда

```
npm run build
```

### Для задания ссылки для проверки XML файла с сервера

Перейдите в

```
src/pages/HomePage.vue
```

и вставьте вместо пустого значения переменной `link` необходимую ссылку
